# Claim Boundary

Research Core v0.1 supports only fixture-level Candidate Foundry behavior:

- typed diagnostic memory ingestion from controlled structured artifacts
- deterministic memory retrieval
- deterministic policy ranking
- selected candidate queue output
- claim-boundary verification

It does not support:

- search-quality improvement
- metric improvement
- formal recommender improvement
- autonomous discovery
- M5 readiness
- runtime-kernel readiness

Any search-quality benchmark, formal evaluation, M5 work, runtime-kernel
extraction, registry promotion, action-space edit, or RecBole core edit requires
a separate user-approved project.
